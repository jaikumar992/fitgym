This is a Streamlit application that provides a simple calorie tracker, exercise library, and progress dashboard.

How to run locally:

1. Activate your virtual environment:
   ```
   .\venv\Scripts\Activate.ps1
   ```
2. Install dependencies (if needed):
   ```
   .\venv\Scripts\python.exe -m pip install -r requirements.txt
   ```
3. Run the app:
   ```
   .\venv\Scripts\streamlit.exe run app.py
   ```

Notes:
- Keep `attached_assets/` in the repository so images are available.
- If you deploy to Streamlit Cloud, push this repository to GitHub and connect it from share.streamlit.io.
